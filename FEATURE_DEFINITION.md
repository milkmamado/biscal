## 📊 1. 시그널 스캐너 (Signal Scanner)

### 핵심 개념
**변동폭 기반 단순 스캐닝** - MTF, ADX, RSI 등 복잡한 기술 분석 제거, 순수 변동폭만으로 종목 선별

### 작동 방식

#### 1차 필터 (기본 조건)
| 모드 | 가격 범위 | 거래량 | 변동폭 |
|------|-----------|--------|--------|
| **잡코인** | $0.01 ~ $1 | $10M+ | 1% ~ 20% |
| **메이저** | 화이트리스트 | $50M+ | 2% ~ 15% |

#### 2차 정렬 (변동성 스코어)
```
변동성 스코어 = 변동폭 점수(0-100) + 거래량 보너스(0-50)
- 최적 변동폭: 3% ~ 8% → 100점
- 거래량: $100M당 50점 보너스 (최대 50점)
```

#### 시그널 생성
- 상위 변동폭 5개 종목 시그널 생성
- 가격 변화율 기준 방향 결정 (양수 → LONG, 음수 → SHORT)
- 변동성 스코어 기준 강도 결정 (80+ = strong, 60+ = medium, else = weak)

### 스캔 주기
- **자동 스캔**: 30초마다
- **수동 스캔**: 버튼 클릭
- **시그널 발견 시**: 자동 일시정지 → 패스하면 재개

## 📈 2. DTFX 차트 표시 (Display Only)

### 핵심 개념
**존 표시 전용** - 자동 스캐닝 기능 제거, 차트에 시각적 참고용으로만 표시

### 표시 요소
- **FVG (Fair Value Gap)**: 가격 불균형 영역
- **OTE (Optimal Trade Entry)**: 0.618 ~ 0.786 피보나치 레벨
- **Premium/Discount Zone**: 고가/저가 영역 구분

### 사용 방식
- 트레이더가 직접 차트 보고 진입 판단
- 자동 매매와 연동되지 않음

## 🤖 3. 자동 매매 (Auto Trading)

### 진입 조건
1. **시그널 스캐너** → 변동폭 시그널 감지
2. **변동폭 필터** → 활성화 시 추가 검증
3. **DTFX 필터** → 활성화 시 OTE 영역 진입만 허용

### 지정가 주문 시스템
```
진입: 현재가 기준 0.02% 유리한 가격
TP: 설정값 (기본 $0.5)
SL: 설정값 (기본 $0.3)
레버리지: 설정값 (기본 10x)
```

### 포지션 관리
- 한 번에 1개 포지션만 보유
- TP/SL 도달 시 자동 청산
- 15분 후 미체결 주문 자동 취소

## 🛠️ 5. 시스템 구조

### 핵심 훅(Hooks)
```
useCoinScreening     → 변동폭 기반 종목 스캐닝
useLimitOrderTrading → 지정가 주문 자동 매매
useDTFX              → 차트 DTFX 존 표시
useTickerWebSocket   → 실시간 시세 수신
```

### 삭제/비활성화된 훅
```
useDTFXScanner       → 삭제됨 (자동 스캔 기능 제거)
useProDirection      → 미사용 (MTF 분석 제거)
useMomentumSignals   → 미사용
useBollingerSignals  → 미사용
useEnhancedSignals   → 미사용
```

## ⚡ 7. 핵심 원칙

### 스캘퍼 철학
1. **단순함** - 복잡한 지표 배제, 변동폭만 추적
2. **속도** - 빠른 스캐닝, 빠른 진입/청산
3. **리스크 관리** - 고정 SL/TP로 손실 제한

### 변동폭 전략
- 높은 변동폭 = 수익 기회
- 거래량 동반 = 신뢰도 상승
- 메이저 코인 = 안정성 우선
