## 🔥 핵심 컨셉

**스캐닝은 자동, 진입은 수동** - 변동폭 높은 종목을 자동 탐지하고, 트레이더가 직접 차트 보고 진입

---

## 📊 1. 시그널 스캐너 (자동 스캐닝)

### 작동 방식

#### 1차 필터
| 모드 | 가격 범위 | 거래량 | 변동폭 |
|------|-----------|--------|--------|
| **잡코인** | $0.01 ~ $1 | $10M+ | 1% ~ 20% |
| **메이저** | 화이트리스트 | $50M+ | 2% ~ 15% |

#### 2차 정렬 (변동성 스코어)
- 최적 변동폭 3~8% → 100점
- 거래량 $100M당 +50점 (최대)

#### 시그널 감지 시
- 차트가 해당 종목으로 자동 전환
- **자동 진입 없음** → 트레이더가 직접 판단
- 30초마다 자동 스캔

## 🎮 3. 수동 진입

### 청산
- SL/TP 설정값 기준 자동 청산
- 수동 청산 버튼

## 🛠️ 5. 시스템 구조

### 핵심 훅(Hooks)
```
useCoinScreening     → 변동폭 기반 종목 스캐닝
useLimitOrderTrading → 지정가 주문 자동 매매
useDTFX              → 차트 DTFX 존 표시
useTickerWebSocket   → 실시간 시세 수신
```

### 삭제/비활성화된 훅
```
useDTFXScanner       → 삭제됨 (자동 스캔 기능 제거)
useProDirection      → 미사용 (MTF 분석 제거)
useMomentumSignals   → 미사용
useBollingerSignals  → 미사용
useEnhancedSignals   → 미사용
```

## ⚡ 7. 핵심 원칙

### 스캘퍼 철학
1. **단순함** - 복잡한 지표 배제, 변동폭만 추적
2. **속도** - 빠른 스캐닝, 빠른 진입/청산
3. **리스크 관리** - 고정 SL/TP로 손실 제한

### 변동폭 전략
- 높은 변동폭 = 수익 기회
- 거래량 동반 = 신뢰도 상승
- 메이저 코인 = 안정성 우선
